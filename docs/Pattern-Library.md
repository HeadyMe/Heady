# Heady Pattern Library

## Cross-Device Synchronization
- **Implementation**: `CrossDeviceSync` component
- **Location**: `headybuddy/src/components/CrossDeviceSync.jsx`
- **Features**:
  - Device listing
  - Active device switching
  - Automatic state sync

## Demo Capabilities
- **Implementation**: `DemoRepos` component
- **Location**: `headybuddy/src/components/DemoRepos.jsx`
- **Features**:
  - Repository listing
  - One-click demo execution
  - Status feedback

## API Documentation Portal
- **Implementation**: Swagger UI + OpenAPI
- **Location**: `docs/api/openapi.yaml` and `heady-manager.js`
- **Features**:
  - Auto-generated documentation
  - Interactive endpoint testing

## Legal IP Protection
- **Implementation**: Header injection + Evidence snapshots
- **Location**: `scripts/legal/inject_headers.py` and `scripts/ops/snapshot_evidence.py`
- **Features**:
  - Automatic copyright headers
  - Patent pending notices
  - Evidence packets for USPTO

## Worktree Cleanup
- **Implementation**: PowerShell script
- **Location**: `scripts/clean-worktrees.ps1`
- **Features**:
  - Stale worktree detection
  - Backup of uncommitted changes
  - Dry run mode
